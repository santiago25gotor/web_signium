# SIGNIUM — Landing Page

Landing page oficial de SIGNIUM, app Android de reconocimiento de lenguaje de señas en tiempo real.

## 📁 Estructura de carpetas

```
signium-web/
├── index.html          ← Página principal
├── css/
│   └── main.css        ← Todos los estilos (variables, componentes, responsive)
├── js/
│   └── main.js         ← Interactividad (idioma, scroll reveal, animaciones)
├── images/
│   ├── signum.png          ← Logo blanco sobre negro
│   ├── signum_negro.png    ← Logo negro (para nav)
│   ├── holiwis.png         ← Foto hero/comunidad (B&W)
│   ├── home_background.jpeg
│   ├── ic_actividad.png    ← Icono misión
│   └── ic_seguridad.png    ← Icono privacidad
└── README.md
```

## 🚀 Deploy en GitHub Pages

1. Sube la carpeta `signium-web/` a un repositorio GitHub
2. Ve a **Settings → Pages**
3. Selecciona la rama `main` y carpeta `/root`
4. ¡Listo! La web estará en `https://tuusuario.github.io/signium-web/`
